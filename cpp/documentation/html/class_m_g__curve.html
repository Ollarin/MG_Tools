Detailed Description

lets you generate a curve out of transform

Author
Marco Giordano
Date
11/13/2012
Version
latest version : V1
changeload versions :
V1 :
  • initial release

node name : MG_curve.

details : This node let you create a nurbs curve using an X number of transform as controlPoints. THe output of this node is not a simple float but a shape , so i suggest you to create a nurbs curve like a circle and then override the connection in the "create" attribute in the curve shape

example create node : (MEL) createNode MG_curve.

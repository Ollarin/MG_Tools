Detailed Description

let s you draw an array of vectors in maya viewport

Author
Marco Giordano
Date
02/11/2013
Version
latest version : V1
changeload versions :
V1 :
  • initial release

node name : MG_vectorGL.

details : This nodelet s you draw an array of vectors in maya viewport. This node is used for debugging purpose , connect in the vecs attribute all the vectors you want to draw and in the startPoits array all the base positions for the vectors, connect the fake out attribute to something in order to force the compute

example create node : (MEL) createNode MG_vectorGL.

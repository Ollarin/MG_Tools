In this repository you can find the source of most of my nodes , I decided to not build the single nodes but to build bundles . There are currently two bundles : MG_toolsLite MG_toolsPro

MG_toolsLite comes with all the plugins with source code and can be recompiled from everyone MG_toolsPro comes only with the build plugin and contains some extra nodes for which I did not release the source
